# Project Context

## Project Name
Kyoto Travel Guide - 京都旅遊指南

## Description
一個介紹京都旅遊資訊的靜態網頁，包含景點推薦、美食推薦、交通資訊與行程規劃。

## Tech Stack
- Frontend: HTML + CSS + JavaScript
- Styling: Tailwind CSS (CDN)
- No backend required (static site)

## Code Conventions
- 使用繁體中文作為主要語言
- 檔案命名使用 kebab-case
- CSS class 使用 Tailwind utility classes

## Architecture
- 單頁式應用（SPA-like）
- 響應式設計（Mobile First）
- 區塊：Hero、景點、美食、交通、行程、Footer
