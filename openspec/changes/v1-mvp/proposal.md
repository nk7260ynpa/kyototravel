# Change Proposal: V1 MVP

## Summary
建立京都旅遊指南網頁的第一個可用版本（MVP）。

## Motivation
需要一個簡潔美觀的靜態網頁，提供京都旅遊的核心資訊，
方便旅客快速查閱景點、美食、交通與行程建議。

## Scope
- 實作所有 REQ-001 ~ REQ-007 的需求
- 使用 Tailwind CSS 進行樣式設計
- 確保 Mobile / Desktop 響應式設計

## Out of Scope
- 後端 API
- 使用者登入系統
- 多語系切換（未來迭代）

## Risks
- 圖片素材需使用免費授權圖庫
- 餐廳資訊可能需定期更新
